TaskFlow API
Project Overview:
TaskFlow API is a backend REST API built using FastAPI to practice backend architecture, authentication, and database persistence. The project implements JWT-based authentication, role-based access, and secure CRUD operations.
This project is designed for learning purposes and is not production-hardened.
Features:
User registration and login
JWT-based authentication
Role-based access control
Task CRUD operations
Modular backend architecture
Tech Stack:
Backend: Python, FastAPI
Authentication: JWT
Database: SQLite (SQLAlchemy ORM)
Validation: Pydantic
Version Control: Git, GitHub

Client

API Request
 ↓
